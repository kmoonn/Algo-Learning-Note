# [104. 二叉树的最大深度](https://leetcode.cn/problems/maximum-depth-of-binary-tree)

## 思路

二叉树最大深度：**根节点**到**最远叶子节点**中间的节点数量。

思考整棵树与其左右子树的关系，`整棵树的最大深度 = max(左子树的最大深度，右子树的最大深度) + 1`， \+ 1 是因为要加上**根节点**。

**递归的核心：原问题、子问题。**

这里的原问题：计算整棵树的最大深度，子问题：计算左/右子树的最大深度。

子问题和原问题是相似的，子问题需要把**计算结果返回给上一级**问题。

接下来思考**递归边界条件**（终止条件）：当递归到**空节点**时，直接返回最大深度 0。

## 解法

### 递归
```java
class Solution {
    public int maxDepth(TreeNode root) {
        if (root == null) {
            return 0;
        }
        
        int left = maxDepth(root.left);
        int right = maxDepth(root.right);
        return Math.max(left, right) + 1;
    }
}
```

### 递归写法 2

维护一个**全局变量**，每次递归传递当前路径节点个数。

```java
class Solution {
    private int ans = 0;

    public int maxDepth(TreeNode root) {
        dfs(root, 0);
        return ans;
    }

    private void dfs(TreeNode node, int cnt) {
        if (node == null) {
            return;
        }

        cnt += 1;
        ans = Math.max(ans, cnt);

        dfs(node.left, cnt);
        dfs(node.right, cnt);
    }
}
```
