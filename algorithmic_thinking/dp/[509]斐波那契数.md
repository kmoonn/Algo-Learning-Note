# [509. 斐波那契数](https://leetcode.cn/problems/fibonacci-number/)

## 解法

### dp

```java
class Solution {
    public int fib(int n) {
        int[] dp = new int[n + 2];
        // dp[0] = 0;
        dp[1] = 1;
        for (int i = 2; i <= n; i++) {
            dp[i] = dp[i - 1] + dp[i - 2];
        }
        return dp[n];
    }
}
```

### 状态压缩

当前状态的值（dp[i]）只依赖于前两个状态的值(dp[i - 1], dp[i - 2])

```java
class Solution {
    public int fib(int n) {
        if (n == 0) return 0;
        int pre = 0;
        int cur = 1;

        for(int i = 2; i <= n; i++) {
            int nxt = pre + cur;
            pre = cur;
            cur = nxt;
        }
        return cur;
    }
}
```
