# [746. 使用最小花费爬楼梯](https://leetcode.cn/problems/min-cost-climbing-stairs)

## 思路

最值问题：**求最小花费**


## 解法

### dp

```java
class Solution {
    
    public int minCostClimbingStairs(int[] cost) {
        int n = cost.length;
        // dp[i] 表示到达第 i 个 台阶所需的最小花费
        int[] dp = new int[n + 1];
        // base case
        dp[0] = 0;
        dp[1] = 0;
        // 穷举
        for (int i = 2; i <= n; i++) {
            dp[i] = Math.min(dp[i - 1] + cost[i - 1], dp[i - 2] + cost[i - 2]);
        }
        return dp[n];
    }
}
```

## 优化

### 状态压缩

状态转移只依赖 dp[i-1] 和 dp[i-2]，完全可以把 dp 数组压缩成两个变量，让空间复杂度从 O(n) 变成 O(1)。

```java
class Solution {
    public int minCostClimbingStairs(int[] cost) {
        int n = cost.length;

        int prev2 = 0; // dp[i - 2]
        int prev1 = 0; // dp[i - 1]

        for (int i = 2; i <= n; i++) {
            int cur = Math.min(prev1 + cost[i - 1], prev2 + cost[i - 2]);
            prev2 = prev1;
            prev1 = cur;
        }

        return prev1; // dp[n]
    }
}
```
