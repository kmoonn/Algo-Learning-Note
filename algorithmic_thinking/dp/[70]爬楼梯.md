# [70. 爬楼梯](https://leetcode.cn/problems/climbing-stairs)

## 思路

爬到第 i 阶的状态，可以从第 i - 1 阶爬 1 个台阶，或者从第 i - 2 阶爬 2 个台阶到达。

## 解法

### dp

```java
class Solution {
    public int climbStairs(int n) {
        int[] dp = new int[n + 2];

        dp[1] = 1;
        dp[2] = 2;

        for (int i = 3; i <= n; i++) {
            dp[i] = dp[i - 1] + dp[i - 2];
        }
        return dp[n];
    }
}
```

## 优化

### 状态压缩

因为当前状态只依赖于前两个状态，所以可以进行空间压缩，仅用两个变量保存前两个状态即可。

```java
class Solution {
    public int climbStairs(int n) {
        if (n < 2) return n;

        int pre = 1;
        int cur = 2;
        int nxt;

        for (int i = 3; i <= n; i++) {
            nxt = pre + cur;
            pre = cur;
            cur = nxt;
        }
        return cur;
    }
}
```
