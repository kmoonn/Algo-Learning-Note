# [216. 组合总和 III](https://leetcode.cn/problems/combination-sum-iii)

## 思路

**组合型**回溯问题。

用一个变量 sum 记录当前所选数字总和。

## 解法

### 回溯

```java
class Solution {

    private List<List<Integer>> ans = new ArrayList<>();

    public List<List<Integer>> combinationSum3(int k, int n) {
        List<Integer> path = new ArrayList<>();
        dfs(n, k, 1, path, 0);
        return ans;
    }

    private void dfs(int n, int k, int start, List<Integer> path, int sum) {
        // 终止条件
        if (path.size() == k) {
            if (sum == n) ans.add(new ArrayList<>(path));
            return;
        }

        // 遍历所有选择
        for (int i = start; i <= 9; i++) {
            // 选择
            path.add(i);
            sum += i;
            // 递归
            dfs(n, k, i + 1, path, sum);
            // 回溯
            path.remove(path.size() - 1);   //path.removeLast();
            sum -= i;
        }
    }
}
```

## 优化

1. 省略 sum 参数传递
2. 剪枝

```java
class Solution {

    private List<List<Integer>> ans = new ArrayList<>();

    public List<List<Integer>> combinationSum3(int k, int n) {
        List<Integer> path = new ArrayList<>();
        dfs(n, k, 1, path);
        return ans;
    }

    private void dfs(int remain, int k, int start, List<Integer> path) {
        // 终止条件
        if (path.size() == k) {
            if (remain == 0) ans.add(new ArrayList<>(path));
            return;
        }

        // 遍历所有选择
        for (int i = start; i <= 9; i++) {
            // 剪枝
            if (i > remain) break;
            // 选择
            path.add(i);
            // 递归
            dfs(remain - i, k, i + 1, path);
            // 回溯
            path.remove(path.size() - 1);   //path.removeLast();
        }
    }
}
```
