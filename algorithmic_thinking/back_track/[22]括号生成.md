# [22. 括号生成](https://leetcode.cn/problems/generate-parentheses)

## 思路

选择：在当前状态可以放 "(" 或 ")"

约束条件：`"(" 数量 < n` 和 `")" 数量 < "(" 已放置数量`

递归：继续放下一个括号

回溯：由于使用 String（不可变），递归参数本身就是独立的，不需要显式回溯。

## 解法

### 回溯
String 是不可变对象

每次 选择 "(" 或 ")" 时，都创建了一个新的 `String path`，原来的 `path` 不会改变。

因此不需要手动回溯撤销。

```java
class Solution {

    private List<String> ans = new ArrayList<>();

    public List<String> generateParenthesis(int n) {
        dfs(n, 0, 0, "");
        return ans;
    }

    // left: 已放置左括号数量
    // right: 已放置右括号数量
    private void dfs(int n, int left, int right, String path) {
        // 终止条件
        if (path.length() == 2 * n) {
            ans.add(path);
            return;
        }

        // ====== 选择 "(" ======
        if (left < n) {
            dfs(n, left + 1, right, path + "(");
        }

        // ====== 选择 ")" ======
        if (right < left) {
            dfs(n, left, right + 1, path + ")");
        }
    }
}
```
