# [39. 组合总和](https://leetcode.cn/problems/combination-sum)

## 思路

组合型回溯

同一个数字可以**无限制重复被选取**

## 解法

### 回溯

注意每个数字可以**重复使用**，所以递归仍然从**当前选择的数字**开始。

```java
class Solution {    
    
    private List<List<Integer>> ans = new ArrayList<>();

    public List<List<Integer>> combinationSum(int[] candidates, int target) {
        dfs(candidates, target, 0, new ArrayList<>(), 0);
        return ans;
    }

    private void dfs(int[] candidates, int target, int start, List<Integer> path, int sum) {
        // 终止条件
        if(sum == target) {
            ans.add(new ArrayList<>(path));
            return;
        }

        // 遍历所有选择
        for (int i = start; i < candidates.length; i++) {
            // 选择
            path.add(candidates[i]);
            sum += candidates[i];
            // 递归
            dfs(candidates, target, i, path, sum);
            // 回溯
            path.removeLast();
            sum -= candidates[i];
        }
    }
}
```

## 优化

### 剪枝

1. 省略 sum 参数
2. 提前终止递归（remain 表示剩余目标值）

```java
class Solution {    
    
    private List<List<Integer>> ans = new ArrayList<>();

    public List<List<Integer>> combinationSum(int[] candidates, int target) {
        dfs(candidates, target, 0, new ArrayList<>());
        return ans;
    }

    private void dfs(int[] candidates, int remain, int start, List<Integer> path) {
        // 终止条件
        if(remain == 0) {
            ans.add(new ArrayList<>(path));
            return;
        }

        // 遍历所有选择
        for (int i = start; i < candidates.length; i++) {
            // 剪枝
            if (candidates[i] > remain) break;
            // 选择
            path.add(candidates[i]);
            // 递归
            dfs(candidates, remain - candidates[i], i, path);
            // 回溯
            path.removeLast();
        }
    }
}
```
