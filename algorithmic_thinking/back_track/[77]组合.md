# [77. 组合](https://leetcode.cn/problems/combinations)

## 思路

**组合型**回溯问题。

回溯模板 **5步** 走：
1. 终止条件
2. 遍历所有选择
3. 选择
4. 递归
5. 回溯

## 解法

### 回溯

注意`ans.add(new ArrayList<>(path));`需要深拷贝。

```java
class Solution {

    private List<List<Integer>> ans = new ArrayList<>();

    public List<List<Integer>> combine(int n, int k) {
        List<Integer> path = new ArrayList<>();
        dfs(n, k, 1, path);
        return ans;
    }

    private void dfs(int n, int k, int start, List<Integer> path) {
        // 终止条件
        if (path.size() == k) {
            ans.add(new ArrayList<>(path));
            return;
        }

        // 遍历所有选择
        for (int i = start; i <= n; i++) {
            // 选择
            path.add(i);
            // 递归
            dfs(n, k, i + 1, path);
            // 回溯
            path.remove(path.size() - 1);   //path.removeLast();
        }
    }
}
```

## 优化

1. 逆序枚举，即 `start = n`，dfs 可以少传一个参数。
2. 剪枝：`i >= k - path.size()`。

```java
class Solution {

    private List<List<Integer>> ans = new ArrayList<>();

    public List<List<Integer>> combine(int n, int k) {
        List<Integer> path = new ArrayList<>();
        dfs(n, k, path);
        return ans;
    }

    private void dfs(int start, int k, List<Integer> path) {
        // 终止条件
        if (path.size() == k) {
            ans.add(new ArrayList<>(path));
            return;
        }

        // 遍历所有选择
        for (int i = start; i >= k - path.size(); i--) {
            // 选择
            path.add(i);
            // 递归
            dfs(i - 1, k, path);
            // 回溯
            path.remove(path.size() - 1);   //path.removeLast();
        }
    }
}
```
