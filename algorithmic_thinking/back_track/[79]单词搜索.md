# [79. 单词搜索](https://leetcode.cn/problems/word-search)

## 思路

**矩阵型（棋盘）回溯**

每个元素都可能作为起点，遍历所有起点。

逐个字符匹配 word。

四个方向递归搜索。

## 解法

### 回溯

```java
class Solution {
    public boolean exist(char[][] board, String word) {
        int m = board.length;
        int n = board[0].length;

        // 每一个元素都可能作为起点
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (dfs(board, m, n, word, i, j, 0))
                    return true;
            }
        }
        return false;
    }

    private boolean dfs(char[][] board, int m, int n, String word, int i, int j, int index) {

        // 终止条件
        if (index == word.length())
            return true; // 匹配完全部字符
        // 边界检查 + 条件判断
        if (i < 0 || i >= m || j < 0 || j >= n || board[i][j] != word.charAt(index))
            return false;

        // 选择
        char c = board[i][j];
        board[i][j] = '#'; // 标记已访问

        // 递归
        boolean found = dfs(board, m, n, word, i + 1, j, index + 1)
                || dfs(board, m, n, word, i - 1, j, index + 1)
                || dfs(board, m, n, word, i, j + 1, index + 1)
                || dfs(board, m, n, word, i, j - 1, index + 1);

        // 回溯
        board[i][j] = c; 
        return found;
    }
}
```
