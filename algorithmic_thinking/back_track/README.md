# 回溯

## 思想

所有回溯算法都使用了**递归**，并且在 **“归”** 阶段执行了关键的 **“回溯”** 操作。

通过试错寻找问题最优解，通常需要遍历所有可能情况。

回溯的核心三部曲：选择——>递归——>回溯
