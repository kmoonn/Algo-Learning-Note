# [93. 复原 IP 地址](https://leetcode.cn/problems/restore-ip-addresses)

## 思路

子集型回溯。

回溯三部曲之前增加了部分条件判断。

## 解法

### 回溯

```java
class Solution {

    private List<String> ans = new ArrayList<>();

    public List<String> restoreIpAddresses(String s) {
        List<String> path = new ArrayList<>();
        dfs(s, 0, path);
        return ans;
    }

    private void dfs(String s, int start, List<String> path) {
        // 终止条件
        if (path.size() == 4) {
            if (start == s.length()) {
                ans.add(String.join(".", path));
            }
            return;
        }
        
        // 遍历所有选择
        for (int i = start; i < s.length(); i++) {
            // 条件判断
            String seg = s.substring(start, i + 1);
            if (seg.length() > 1 && seg.charAt(0) == '0') break;
            if (Integer.parseInt(seg) > 255) break;

            // 选择
            path.add(seg);
            // 递归
            dfs(s, i + 1, path);
            // 回溯
            path.remove(path.size() - 1);
        }
    }
}
```

## 优化

剪枝：当剩余字符数量不够或者过多时，提前剪枝。

```java
private void dfs(String s, int start, List<String> path) {
        // 剪枝
        int remain = s.length() - start;
        int need = 4 - path.size();
        if (remain < need || remain > need * 3) return;

        if (path.size() == 4) {
            if (start == s.length()) {
                ans.add(String.join(".", path));
            }
            return;
        }

        for (int i = start; i < s.length(); i++) {
            String seg = s.substring(start, i + 1);

            if (seg.length() > 1 && seg.charAt(0) == '0') break;
            if (Integer.parseInt(seg) > 255) break;

            // 选择
            path.add(seg);
            // 递归
            dfs(s, i + 1, path);
            // 回溯
            path.removeLast();
        }
}
```
