# [17. 电话号码的字母组合]()

## 思路

多重循环（循环次数不确定） ——> 递归

path 数组：记录每次选择字母后组成的字符串。

递归入口：从 path 第一个位置开始

终止条件：按键次数用完（`i == digits.length`）

选择：在对应数字按键所表示的多个字母选择一个

递归：确定下一个位置（`i+1`）的字母

## 解法

### 递归

隐式回溯撤销

```java
class Solution {
    private static final String[] MAPPING = new String[] { "", "", "abc", "def", "ghi", "jkl", "mno", "pqrs", "tuv",
            "wxyz" };

    public List<String> letterCombinations(String digits) {
        int n = digits.length();

        List<String> ans = new ArrayList<>();
        char[] path = new char[n];
        dfs(0, ans, path, digits.toCharArray());
        return ans;
    }

    private void dfs(int i, List<String> ans, char[] path, char[] digits) {
        if (i == digits.length) {
            ans.add(new String(path));
            return;
        }

        String letters = MAPPING[digits[i] - '0'];
        for (char c : letters.toCharArray()) {
            // 每次循环 path[i] 被覆盖（隐式回溯）
            path[i] = c;
            dfs(i + 1, ans, path, digits);
        }
    }
}
```

### 回溯

```java
class Solution {
    private static final String[] map = new String[] { "", "", "abc", "def", "ghi", "jkl", "mno", "pqrs", "tuv", "wxyz" };

    public List<String> letterCombinations(String digits) {
        List<String> res = new ArrayList<>();

        backtrack(res, digits, 0, new StringBuilder());
        return res;
    }

    private void backtrack(List<String> res, String digits, int i, StringBuilder path) {
        // 终止条件
        if (i == digits.length()) {
            res.add(path.toString());
            return;
        }
        
        // 准备所有的选择
        String letters = map[digits.charAt(i) - '0'];
        for (char c : letters.toCharArray()) {
            // 选择
            path.append(c);
            // 递归
            backtrack(res, digits, i + 1, path);
            // 回溯
            path.deleteCharAt(path.length() - 1);
        }
    }
}
```
