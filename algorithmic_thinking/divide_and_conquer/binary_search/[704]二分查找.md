# [704. 二分查找](https://leetcode.cn/problems/binary-search)

## 思路

经典二分

二分查找某一个具体的答案

## 解法

### 二分查找

**闭区间** 写法

```java
class Solution {
    public int search(int[] nums, int target) {
        // 闭区间
        int left = 0;
        int right = nums.length - 1;
        // 终止条件
        while (left <= right) {
            // 防止加法溢出
            int mid = left + (right - left) / 2;
            // 缩小查找区间
            if (nums[mid] < target) {
                left = mid + 1;
            } else if (nums[mid] > target) {
                right = mid - 1;
                // 二分答案，找到即返回
                // nums[mid] == target
            } else {
                return mid;
            }
        }
        return -1;
    }
}
```
