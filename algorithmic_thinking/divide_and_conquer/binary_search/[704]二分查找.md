# [704. 二分查找](https://leetcode.cn/problems/binary-search)

## 思路

经典二分

找某一个具体的答案（目标值）

## 解法

### 二分查找

**闭区间** 写法

```java
class Solution {
    public int search(int[] nums, int target) {
        // 闭区间
        int left = 0;
        int right = nums.length - 1;
        // 终止条件
        while (left <= right) {
            // 防止加法溢出
            int mid = left + (right - left) / 2;
            // 缩小查找区间
            if (nums[mid] < target) {
                left = mid + 1;
            } else if (nums[mid] > target) {
                right = mid - 1;
                // 二分答案，找到即返回
                // nums[mid] == target
            } else {
                return mid;
            }
        }
        return -1;
    }
}
```

**开区间** 写法

```java
class Solution {
    public int search(int[] nums, int target) {
        // 开区间
        int left = -1;
        int right = nums.length;
        // 终止条件（保证区间内有值）
        while (left + 1 < right) {
            // 防止加法溢出
            int mid = left + (right - left) / 2;
            // 缩小查找区间
            if (nums[mid] < target) {
                left = mid;
            } else if (nums[mid] > target) {
                right = mid;
            // 二分答案，找到即返回
            // nums[mid] == target
            } else {
                return mid;
            }
        }
        return -1;
    }
}
```

**左闭右开区间** 写法

```java
class Solution {
    public int search(int[] nums, int target) {
        // 左闭右开区间
        int left = 0;
        int right = nums.length;
        // 终止条件（保证区间内有值）
        while (left < right) {
            // 防止加法溢出
            int mid = left + (right - left) / 2;
            // 缩小查找区间
            if (nums[mid] < target) {
                left = mid + 1;
            } else if (nums[mid] > target) {
                right = mid;
            // 二分答案，找到即返回
            // nums[mid] == target
            } else {
                return mid;
            }
        }
        return -1;
    }
}
```


## 优化

- 移位法代替除法提高效率

`int mid = (left + right) >>> 1;`
