# [69. x 的平方根](https://leetcode.cn/problems/sqrtx)

## 思路

找第一个平方后大于等于 target 的值 - 1，就是 `floor(sqrt(x))`。

## 解法

### 二分查找

**闭区间**写法

```java
class Solution {
    public int mySqrt(int x) {
        if (x < 2)
            return x;

        int ans = lowerBound(1, x / 2, (long) x + 1);
        return ans - 1;
    }

    public int lowerBound(int left, int right, long target) {
        while (left <= right) {
            int mid = left + (right - left) / 2;
            long square = (long) mid * mid;

            if (square < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        return left;
    }
}
```

**开区间**写法

```java
class Solution {
    public int mySqrt(int x) {
        if (x < 2)
            return x;

        int ans = lowerBound(1, x / 2 + 1, (long) x + 1);
        return ans - 1;
    }

    public int lowerBound(int left, int right, long target) {
        while (left + 1 < right) {
            int mid = (left + right) >>> 1;
            long square = (long) mid * mid;

            if (square < target) {
                left = mid;
            } else {
                right = mid;
            }
        }
        return right;
    }
}
```

## 注意

优化点：区间右边界可以为` target / 2`。

需要使用的 long 类型确保精度

因为 mid * mid 在 int 范围内可能溢出（当 mid ≥ 46341 时），而 long 可以覆盖最大到 2^63-1，足够安全。
