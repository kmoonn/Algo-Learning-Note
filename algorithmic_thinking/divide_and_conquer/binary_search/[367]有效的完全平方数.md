# [367. 有效的完全平方数](https://leetcode.cn/problems/valid-perfect-square)

## 思路

二分查找第一个**平方后**大于等于 target 的数

## 解法

### 二分查找

**闭区间**写法

```java
class Solution {
    public boolean isPerfectSquare(int num) {
        int ans = lowerBound(1, num / 2, num);
        return ans * ans == num; 
    }

    public int lowerBound(int left, int right, long target) {
        while (left <= right) {
            int mid = left + (right - left) / 2;
            long square = (long) mid * mid;

            if (square < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        return left;
    }
}
```

**开区间**写法

```java
class Solution {
    public boolean isPerfectSquare(int num) {
        int ans = lowerBound(0, num / 2 + 1, num);
        return ans * ans == num; 
    }

    public int lowerBound(int left, int right, long target) {
        while (left + 1 < right) {
            int mid = (left + right) >>> 1;
            long square = (long) mid * mid;

            if (square < target) {
                left = mid;
            } else {
                right = mid;
            }
        }
        return right;
    }
}
```

## 优化

- 区间右边界可以设置为 `num / 2`。
