# [153. 寻找旋转排序数组中的最小值](https://leetcode.cn/problems/find-minimum-in-rotated-sorted-array)

## 思路

旋转后数组被分为两段有序段。

前一段最小值大于第二段最大值。

最小值一定是第 2 段的第一个元素。

## 解法

### 暴力

逐个遍历，更新最小值。

### 二分查找

```java
class Solution {
    public int findMin(int[] nums) {
        int n = nums.length;
        int left = 0, right = n - 1;

        // 当区间收缩到只剩一个元素时（left = right），该元素即为最小值。
        while (left < right) {
            int mid = left + (right - left) / 2;
            // 收缩右边界（mid 落在第 2 段）
            if (nums[mid] < nums[right]) {
                right = mid;
                // 收缩左边界（mid 落在第 1 段）
            } else if (nums[mid] > nums[right]) {
                left = mid + 1;
            }
            // 题目要求不存在重复元素，即不存在 nums[mid] = nums[right]
        }
        return nums[left];
        // return nums[right], 最后 left = right
    }
}
```

### 快速排序

快速排序后取第一个数。
