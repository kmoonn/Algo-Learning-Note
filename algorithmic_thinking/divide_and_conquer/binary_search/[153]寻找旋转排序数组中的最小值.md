# [153. 寻找旋转排序数组中的最小值](https://leetcode.cn/problems/find-minimum-in-rotated-sorted-array)

## 思路

旋转后数组被分为两段有序段。

## 解法

### 二分查找

```java
class Solution {
    public int findMin(int[] nums) {
        int n = nums.length;
        int left = 0, right = n - 1;

        while (left < right) {
            int mid = left + (right - left) / 2;
            // 收缩右边界
            if (nums[mid] < nums[right]) {
                right = mid;
                // 收缩左边界
            } else {
                left = mid + 1;
            }
        }
        return nums[left];
    }
}
```
