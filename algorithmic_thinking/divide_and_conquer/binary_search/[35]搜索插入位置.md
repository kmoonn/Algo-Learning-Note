# [35. 搜索插入位置](https://leetcode.cn/problems/search-insert-position)

## 思路

二分查找

查找第一个 ≥ target 的元素，返回其索引。

循环不变量：left 指针左边元素都小于 target，right 指针右边元素都 ≥ target。

因此**循环结束后**，left 指向第一个 ≥ target 的元素。

## 解法

### 二分查找

**闭区间** 写法

```java
class Solution {
    public int searchInsert(int[] nums, int target) {
        int left = 0;
        int right = nums.length - 1;
        while (left <= right) {
            int mid = left + (right - left) / 2;
            if (nums[mid] < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        return left;
    }
}
```
