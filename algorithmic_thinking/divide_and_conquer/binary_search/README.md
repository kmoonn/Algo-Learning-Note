# 二分查找

## 思想

**循环不变量**

1. left 指针左边的元素都小于 target
2. right 指针右边的元素都大于等于 target

## 框架

1. 定义首尾指针
2. 循环终止条件
3. 缩小查找区间
4. 返回答案

### 分类

* 找目标值
* 找左边界
* 找右边界
* 找第一个大于等于

## 优化

- 防止整数溢出
  - 移位运算 `>>>`
