### 二分查找

```java
class Solution {
    public int findMin(int[] nums) {
        int n = nums.length;
        int left = 0, right = n - 1;

        // 当区间收缩到只剩一个元素时（left = right），该元素即为最小值。
        while (left < right) {
            int mid = left + (right - left) / 2;
            // 收缩右边界（mid 落在第 2 段）
            if (nums[mid] < nums[right]) {
                right = mid;
                // 收缩左边界（mid 落在第 1 段）
            } else if (nums[mid] > nums[right]) {
                left = mid + 1;
                // 安全缩小范围（无法判断 mid 落在哪一段）
            } else {
                right--;
            }
        }
        return nums[right];
    }
}
```
