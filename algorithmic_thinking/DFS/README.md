# 深度优先搜索

## 思想

| 特征      | DFS（深度优先遍历）     | 回溯（Backtracking） |
|---------|-----------------|------------------|
| 目标      | 遍历或构造结构（图、树、路径） | 搜索所有可能的解         |
| 状态变化    | 每条路径走一次，不撤销     | 每次尝试选择、撤销选择      |
| 是否枚举所有解 | ❌ 只构造唯一确定路径     | ✅ 要枚举并筛选所有可能     |
| 是否需要剪枝  | 通常不需要           | 常常需要             |

## 框架

### Hierholzer 算法

用于求有向图或无向图的欧拉路径（Eulerian Path）。

算法流程是：
1. 从起点出发； 
2. 一直沿未走过的边前进； 
3. 走到无法前进的节点时，将该节点加入结果； 
4. 递归返回时继续补全路径。
