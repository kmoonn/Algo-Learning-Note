## 解法

### 排序 + 双指针

```java
class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        // 排序
        Arrays.sort(nums);

        List<List<Integer>> ans = new ArrayList<>();
        int n = nums.length;

        // 固定第一个数
        for (int i = 0; i < n - 2; i++) {
            int x = nums[i];
            // 跳过重复元素
            if (i > 0 && nums[i - 1] == x)
                continue;

            // 首尾指针
            int j = i + 1;
            int k = n - 1;
            while (j < k) {
                int s = x + nums[j] + nums[k];
                // 收缩指针
                if (s > 0)
                    k--;
                else if (s < 0)
                    j++;
                // s == 0
                else {
                    ans.add(List.of(x, nums[j], nums[k]));
                    // 跳过重复 j
                    while (j < k && nums[j + 1] == nums[j])
                        j++;
                    // 跳过重复 k
                    while (j < k && nums[k - 1] == nums[k])
                        k--;
                    // 收缩指针
                    j++;
                    k--;
                }
            }
        }
        return ans;
    }
}
```
