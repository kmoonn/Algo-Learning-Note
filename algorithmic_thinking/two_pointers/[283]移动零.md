## 解法

### 双指针

```java
class Solution {
    public void moveZeroes(int[] nums) {
        // 左指针：标记下一个非零元素应该放置的位置（初始在数组开头）
        int left = 0;
        
        // 右指针：遍历整个数组，寻找非零元素
        for (int right = 0; right < nums.length; right++) {
            // 核心判断：右指针遇到非零元素
            if (nums[right] != 0) {
                // 交换左、右指针位置的元素：把非零元素放到左指针位置
                int tmp = nums[right];
                nums[right] = nums[left];
                nums[left] = tmp;
                
                // 左指针右移：下一个非零元素要放到下一个位置
                left++;
            }
            // 如果右指针遇到0，直接跳过，左指针不动
        }
    }
}
```
