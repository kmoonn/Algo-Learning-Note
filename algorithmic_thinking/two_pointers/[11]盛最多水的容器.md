## 解法

### 双指针 + 贪心

```java
class Solution {
    public int maxArea(int[] height) {
        // 1. 初始化双指针：左指针在数组开头，右指针在数组末尾
        int left = 0;
        int right = height.length - 1;

        // 2. 记录最大面积，初始为0
        int ans = 0;

        // 3. 双指针向中间收缩，直到相遇
        while (left < right) {
            // 计算容器的水平宽度（右指针 - 左指针）
            int w = right - left;
            // 计算容器的有效高度（取两侧高度的较小值，因为水会从低的一侧溢出）
            int h = Math.min(height[left], height[right]);
            // 计算当前容器的面积
            int area = w * h;
            // 更新最大面积
            ans = Math.max(ans, area);

            // 4. 核心策略：移动较短的一侧指针
            if (height[left] < height[right]) {
                left++; // 左指针短，右移左指针
            } else {
                right--; // 右指针短（或相等），左移右指针
            }
        }
        return ans;
    }
}
```
