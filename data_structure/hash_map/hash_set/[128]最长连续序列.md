## 解法

### 哈希集合

```java
class Solution {
    public int longestConsecutive(int[] nums) {
        // 边界条件：空数组直接返回0
        if (nums.length == 0)
            return 0;
        
        // 1. 用HashSet去重，同时提供O(1)时间复杂度的查找
        HashSet<Integer> set = new HashSet<>();
        for (int num : nums) {
            set.add(num);
        }

        // 2. 记录最长连续序列的长度
        int ans = 0;

        // 3. 遍历去重后的所有数字
        for (int num : set) {
            // 核心判断：只有当前数字是“序列起点”（num-1不存在）时，才开始统计
            if (!set.contains(num - 1)) {
                int curNum = num; // 当前遍历到的数字（从起点开始）
                int count = 1;    // 统计当前序列长度，至少包含起点本身

                // 4. 不断查找下一个连续数（curNum+1），直到找不到为止
                while (set.contains(curNum + 1)) {
                    curNum++;    // 往后移动一位
                    count++;     // 序列长度+1
                }
                
                // 5. 更新最长序列长度
                ans = Math.max(ans, count);
            }
        }

        return ans;
    }
}
```
